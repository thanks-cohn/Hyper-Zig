# Hyper-Zig Roadmap

Hyper-Zig is an experimental Zig 0.14.x RISC-V hypervisor project.

Current state:
- No Linux guest support yet.
- No BusyBox boot yet.
- No guest execution yet.
- No HGATP write yet.
- No active second-stage translation yet.

Use `COMMANDS.md` for current commands.
Use `docs/MILESTONE_INDEX.md` for milestone evidence.
Use `./scripts/validate-hyperzig.sh` for validation.
