# AI Assistance Policy

ZIGN01D permits AI assistance.

AI-generated or AI-assisted code must be reviewed by a human maintainer before merge.

AI assistance must not be used to fake smoke test results.

AI assistance must not claim implemented capability without proof.

Documentation must distinguish implemented, scaffolded, and not-implemented features.

Smoke tests are the source of truth for capability claims.

Generated transcripts/logs should not be committed unless intentionally added as fixtures.

Maintainers are responsible for final code and claims.
